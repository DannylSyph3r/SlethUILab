# SlethUILab

A curated collection of experimental interfaces and visual effects.

## Effects

### Fractal Glass

Aurora-like flowing waves behind ribbed glass with customizable colors and dynamics. Features WebGL-powered animations with real-time settings control.

### Snow Effect

Realistic particle-based snowfall with depth simulation, mouse avoidance, and dynamic wind.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Graphics**: WebGL / Canvas API
- **UI Components**: Radix UI

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the project.
